# U.S. Census Data Cleaning and Analysis

This project focuses on cleaning, transforming, and preparing U.S. Census state-level demographic data for analysis. The dataset includes population, income, gender, and race-related information across U.S. states and territories.


## Project Overview

This project focuses on cleaning and transforming U.S. Census state-level demographic data to make it suitable for analysis. The dataset was processed to handle missing values, standardize formats, and extract structured information from raw columns such as income and gender population data.


## Dataset

The dataset includes the following fields:

- `State`
- `TotalPop`
- `Hispanic %`
- `White %`
- `Black %`
- `Native %`
- `Asian %`
- `Pacific %`
- `Income`
- `GenderPop`
- `Men Count`
- `Women Count`


## Data Processing

The dataset was cleaned and transformed to make it analysis-ready.  
Key operations included merging multiple CSV files, handling missing values, converting data types, and extracting structured information from mixed-format columns.


## Notes

Some missing values were intentionally left unfilled in certain columns so they can be evaluated as possible indicators of data quality issues or incomplete reporting.
